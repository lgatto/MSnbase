# Update MSnbase objects

Methods for function `updateObject` for objects from the MSnbase
package. See
[`updateObject`](https://rdrr.io/pkg/BiocGenerics/man/updateObject.html)
for details.

## Methods

- `signature(object = "MSnExp")`:

  Update the `MSnExp` object to the latest class version

- `signature(object = "Spectrum")`:

  Update the `Spectrum` object (and it's sub-classes `Spectrum1` and
  `Spectrum2`) to the latest class version.
